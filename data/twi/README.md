# TWI SIG Mailing-List Archive

Public archive of the [Confidential Computing Trustworthy Workload Identity SIG](https://lists.confidentialcomputing.io/g/Trustworthy-Workload-Identity-SIG) mailing list, scraped from groups.io and rendered as a browsable wiki.

## Start here

- **[Wiki overview](wiki/overview.md)** — the curated synthesis: concepts, entities, key findings, and a 13-month timeline.
- **[Wiki timeline](wiki/timeline.md)** — chronological narrative of the SIG.
- **[Wiki log](wiki/log.md)** — record of ingests / lint passes against the wiki.

## Source threads

Every other `*.md` file at this level is one mailing-list thread, named `<topic-id>-<slug>.md`. They contain raw message bodies in chronological order with frontmatter (subject, participants, date range, message count, source URL).

The wiki pages cite these source threads via footnotes. Click any citation badge in a wiki page to open the underlying thread.
