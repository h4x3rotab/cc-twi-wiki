# linux-coco Mailing List Archive

Upstream Linux kernel mailing list for confidential computing (`linux-coco@lists.linux.dev`), cloned from the public-inbox mirror at [lore.kernel.org/linux-coco](https://lore.kernel.org/linux-coco).

**Coverage:** May 2024 – May 2026 · 602 threads · 9,235 messages

→ **[overview.md](overview.md)** — hub page with key findings, topic map, and links to all wiki pages.

- [Intel TDX](concepts/tdx.md) — TDX Module Update (v8 → kernel 7.2), Dynamic PAMT, kexec, MMIO
- [AMD SEV-SNP](concepts/sev-snp.md) — RMPOPT, firmware hotloading, attestation, SVSM
- [ARM CCA](concepts/arm-cca.md) — KVM Realm support (RMM v2.0-beta), guest driver, LFA
- [PCI/TDISP](concepts/pci-tdisp.md) — Cross-vendor PCIe device attestation framework
- [guest_memfd](concepts/guest-memfd.md) — In-place shared/private conversion (v6)
- [SVSM](concepts/svsm.md) — Secure VM Service Module (COCONUT-SVSM, vTPM)
- [TSM Framework](concepts/tsm-framework.md) — Kernel's cross-vendor CoCo API
