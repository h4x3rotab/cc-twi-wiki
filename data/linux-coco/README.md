# linux-coco Mailing List Archive

This directory contains the **linux-coco** mailing list archive (`linux-coco@lists.linux.dev`) — the upstream Linux kernel mailing list for confidential computing — cloned from the public-inbox mirror at [lore.kernel.org/linux-coco](https://lore.kernel.org/linux-coco).

**Coverage:** May 2025 – May 2026 · 331 threads · 5,482 messages

Each `YYYYMMDD-slug.md` file is one thread (all messages in the thread, oldest first).
